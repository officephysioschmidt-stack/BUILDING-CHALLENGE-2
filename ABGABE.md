# Vorlage für deinen Abgabe-Post

Der Post gehört am Ende der Challenge in die Kategorie "Building Challenge"
in der SKAILE Academy. Einfach kopieren und Platzhalter ersetzen.

## Abgabe-Post (bis zur Deadline)

```
ABGABE — [Projektname]

Problem: [1 Satz — wessen Problem, was kostet es heute?]
Demo: [Loom- oder YouTube-unlisted-Link — ein Durchlauf, ungeschnitten]
Repo: [GitHub-Link]
Neu in der Challenge entstanden: [2-3 Stichpunkte]
Stack: [Claude Code / n8n / beides]
```

**Zum Demo-Video:** Bildschirmaufnahme reicht völlig (Mac: Cmd+Shift+5).
Keine Kamera, keine Stimme nötig, kein Schnitt erlaubt — gezeigt wird EIN
Durchlauf: Input rein, Agent arbeitet, Ergebnis raus. Niemand bewertet
deine Video-Qualität, nur deinen Agenten.

**Vertrauliches Projekt?** Wenn dein Projekt vertrauliche Kundeninhalte
zeigt, schick die Abgabe stattdessen per DM an Sebastian — gleiche Vorlage,
gleiche Deadline. Dein Repo kannst du privat stellen und Sebastian als
Collaborator einladen (Settings → Collaborators → `sebaskauf`).
